Some of the funniest original content you'll find is on Reddit's r/PhotoShopBattles. Unfortunately, to look at the different photoshops of an image takes a lot of work. I looked to solve this problem by builidng a website that congregates the photoshops for an easier viewing experience. You can check it out here. I wrote the frontend with React and the backend with Node.js. It is hosted on Heroku and uses a PostgreSQL database.

At the 2019 Microsoft hackathon I worked with a team to build a project called Trail Smarts. Our project used machine learning to build unique hiking profiles based on users GPS data from hikes. With these profiles, a React.js frontend - which I helped build - could visualize how easy a particular hike would be for you and how well you would pair with other hikers on that hike.

For the final project of CMPUT 275 I worked with a partner to build a Sodoku solver visualizer. Numerous different solving algorithms were written in C++. For the visualizing component a UI was written to run on an Arduino and an over the wire format was used to send visualization data between the UI and solving backend.
